Keyword

All browsers support 140 different colors by keyword.

RGB

All browsers support color by red green blue (RGB) values. The format for using this is rgb(red, green, blue). The intensity of each color ranges from 0 to 255 allowing 16.7 million color choices.

Hexadecimal

All browsers support color by Hexadecimal (HEX) values. The format for using this is #RRGGBB. RR (red), GG (green) and BB (blue) are hexadecimal integers between 00 and FF specifying the intensity of the color.

HSL

Modern browsers support HSL. The format for using this is hsl(hue, saturation, lightness). Hue is a color wheel so has 360 degrees where 0 is red, 120 is green and, 240 is blue. Saturation ranges from monochrome at 0% to full color at 100%. Lightness ranges from dark at 0% to white at 100%. A lightness of 50% would be used for a solid color.
